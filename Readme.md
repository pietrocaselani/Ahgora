Ahgora
===

Ahgora is:

* a wrapper arround the unofficial Ahgora's API
* a iOS app to interact with Ahgora.

## Instructions

Run the following commands

* `cd Ahgora && pod install`

* `Open Ahgora.xcworkspace`
